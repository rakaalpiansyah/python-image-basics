# 🚀 Dasar Pengolahan Citra dengan Python (OpenCV)
[![Python Version](https://img.shields.io/badge/python-3.x-blue.svg)](https://www.python.org/)
[![OpenCV](https://img.shields.io/badge/OpenCV-4.x-green.svg)](https://opencv.org/)
[![Jupyter](https://img.shields.io/badge/Jupyter-Notebook-orange.svg)](https://jupyter.org/)

> Repositori ini berisi implementasi fundamental pengolahan citra digital menggunakan Python. Proyek ini mendemonstrasikan cara membaca, memanipulasi ruang warna, dan menganalisis distribusi piksel pada gambar.

## ✨ Fitur Utama
- **🎛️ Manipulasi Ruang Warna:** Konversi citra dari BGR (bawaan OpenCV) ke format RGB dan *Grayscale*.
- **📐 Transformasi Dasar:** Memotong (*Cropping*) area spesifik dan mengubah resolusi (*Resizing*) gambar.
- **📊 Analisis Visual:** Visualisasi distribusi intensitas piksel melalui grafik Histogram warna (R, G, B) dan *grayscale*.
